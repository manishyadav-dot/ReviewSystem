New developer setup and team onboarding guide.

## Setup Checklist

- [ ] Clone the repository
- [ ] Install frontend dependencies (`src/frontend/`)
- [ ] Install .NET 10 SDK and restore backend packages (`src/backend/`)
- [ ] Configure local database connection
- [ ] Run database migrations (`src/database/migrations/`)
- [ ] Start backend API and frontend dev server
- [ ] Run test suites to verify setup (`tests/`)
- [ ] Read `Standards/git-workflow.md`
