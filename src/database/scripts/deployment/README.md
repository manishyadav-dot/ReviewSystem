# Deployment Scripts

Scripts for initial database provisioning and environment setup.

## Examples (planned)

- `create-database.sql` — Create the application database
- `create-login.sql` — Create application service account (no passwords in repo)
- `grant-permissions.sql` — Apply least-privilege role permissions
- `run-migrations.sql` — Execute all pending migrations in order

Run these scripts as part of the environment provisioning process documented in `docs/deployment/`.
