# Maintenance Scripts

Scheduled and on-demand database maintenance operations.

## Examples (planned)

- `rebuild-indexes.sql` — Rebuild fragmented indexes
- `update-statistics.sql` — Refresh query optimizer statistics
- `purge-old-data.sql` — Archive or delete expired records
- `health-check.sql` — Database health validation queries

Schedule recurring maintenance via SQL Server Agent jobs or DevOps pipelines.
