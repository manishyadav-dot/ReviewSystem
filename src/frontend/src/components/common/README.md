# Common Components

Generic, reusable UI primitives used across multiple features.

## Examples (planned)

- `Button` — Primary, secondary, and icon button variants
- `Input` — Text, number, date, and search inputs with validation states
- `Modal` — Accessible dialog overlay
- `DataTable` — Sortable, paginated data grid
- `Spinner` — Loading indicator
- `Alert` — Inline notification banners

Keep components composable and free of business logic.
